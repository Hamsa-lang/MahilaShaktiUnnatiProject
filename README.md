Mahila Shakti Unnati – Digital Ledger for Women SHGs

📌 Project Overview

Mahila Shakti Unnati is an Android application developed for women Self-Help Groups (SHGs) to digitally manage savings, loans, repayments, and member records. The app replaces traditional paper registers with a secure and transparent digital system.

This project was developed as part of the MindMatrix VTU Internship Program.

---

🚀 Features

- 👩 Member Registration and Management
- 💰 Weekly Savings Tracking
- 📊 Loan Management System
- 📈 Simple Interest Calculation
- ✅ Loan Validation for unpaid loans
- 📂 Offline Data Storage using Room Database
- 📱 Android-based user interface
- 🔄 MVVM Architecture
- 📤 Report Sharing Support

---

📱 Modules

1. Member Management

- Add members
- Store member details
- View member list

2. Savings Module

- Record weekly savings
- Track paid/pending status
- Calculate total savings

3. Loan Module

- Issue loans
- Track repayments
- Prevent multiple unpaid loans

4. Reports

- Generate summaries
- Share reports

---

🧮 Database Tables

MemberEntity

Stores member details.

SavingsEntity

Stores savings entries for members.

LoanEntity

Stores loan and repayment information.
